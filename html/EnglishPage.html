Some mistakes you've ever had

on bus -> on the bus

If you have only one day in life... -> If you had only one...
虚拟语气与现在事实相反的情况
从句：If+主语+过去时(Be动词用were)
主句：主语+ should/would/might/could+do

listenning -> listening(这个并不是“辅音-元音-辅音”双写最后的辅音+ing的情况)

sometime: 某时（过去或将来）
sometimes: 有时

a good enough girl -> a girl good enough
